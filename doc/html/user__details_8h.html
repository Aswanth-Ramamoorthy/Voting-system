int user_details ( char *  name)

This function is to display the user details from the file.

This function displays the available user information such as First name ,Last name, Age,Phone Number ,postal code and Province.

It compares the given input(i.e Username) with the existing data in the file.

Parameters
[in]Thefunction takes the User name as the input parameter.
[out]Thefunction calls the candidate_details function and displays the candidates in the user's province.
Returns
Returns 0 if the username matches with the one in file, Returns 1 if name is not valid in file
